## post_pubapi

### Example Usage

```python
import os
import ttd_workflows
from ttd_workflows import Workflows


with Workflows(
    ttd_auth=os.getenv("WORKFLOWS_TTD_AUTH", ""),
) as workflows:

    res = workflows.pub_api.post_pubapi(request={
        "method_type": ttd_workflows.PubAPIMethodType.DELETE,
        "endpoint": "<value>",
        "data_body": None,
    })

    # Handle response
    print(res)

```
