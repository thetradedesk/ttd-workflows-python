## post_graphql

An endpoint that executes valid GraphQL queries.

### Example Usage

```python
import os
from ttd_workflows import Workflows


with Workflows(
    ttd_auth=os.getenv("WORKFLOWS_TTD_AUTH", ""),
) as workflows:

    res = workflows.graph_ql.post_graphql()

    # Handle response
    print(res)

```
